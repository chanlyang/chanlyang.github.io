Field[] getDeclaredFields() ；返回 Field物体反射所有字段的类或接口的 类对象表示声明数组。 
    一个Field提供有关信息和动态访问，一个类或接口的一个单一的领域。反射场可以是一个类（静态）字段或一个实例字段。
方法[] getDeclaredMethods() ;
    返回一个数组包含 方法物体反射所有声明的方法的类或接口的类对象，代表包括公众、保护，默认（包）的访问，和私有方法，但不包括继承的方法。   

方法 getMethod(String name, 类<?>... parameterTypes) ;
    返回一个 方法对象反映的类或接口的 类对象表示的指定公共成员方法。  
T newInstance() ;创建这个 类对象表示的类的一个新实例。  
Constructor<T> getDeclaredConstructor(类<?>... parameterTypes) ；返回一个        
                Constructor对象反映指定的构造函数的类或接口的 类对象表示。 
动态创建对象：
    Class aa = Dog.class;
    Object dog = aa.newInstance();  //无参
    Object dog = aa.getDeclaredConstructor(String.class).newInstance("旺财"); //有参
    Method mm = aa.getDeclaredMethod("playDisc",String.class);
    mm.invoke(dog,"椭圆");    //动态调用对象的行为
