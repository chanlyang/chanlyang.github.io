C++与Java平台对比：
2000年左右，java逐渐把c++替掉了

C++有指针   好处： 速度快
            缺陷： 内存溢出、内存泄漏
            new    分配内存
            delete 释放内存
变量：全局变量（C语言）  定义C语言函数外面的变量
      成员变量（C++）  定义在类的内部，方法的外部
              在构造函数分配内存，在析构函数释放内存
      局部变量 ： 定义在函数或方法内
              在函数内分配内存，在函数内释放内存

Java没有内存泄漏：GC 垃圾回收机制
JVM中有堆，虚拟机栈，本地方法栈，程序计数器等，方法区（运行期常量池）--static变量，字符型常量String，Class类型加载。

java7,将static移入heap中
java8，将字符床常量移入heap中

Java是半编译程序——>最终解释执行的
.java编译成.class，字节码（与CPU无关）
javascript是纯解释性语言（newscape网景公司）
